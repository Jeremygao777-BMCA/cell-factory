# Cell-to-Protein Demo Game

An interactive, game-like demonstration of **DNA replication → transcription → translation**.

## Quick start

- Open `index.html` directly in a browser (should work), or run a tiny local server:
  - `python3 -m http.server 8000`
  - Visit `http://localhost:8000`

## What’s included

- `docs/GAME_DESIGN.md`: full design, modes, visuals, mechanics
- `docs/NEXT_AI_HANDOFF.md`: specific tasks for the next AIs
- `index.html` + `src/`: a minimal playable prototype (no dependencies)
