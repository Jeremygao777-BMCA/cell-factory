# Demo Script (5–7 minutes)

## 1) Hook (30s)
- “Cells turn DNA instructions into proteins. Let’s play the process.”

## 2) DNA customization (60s)
- Paste/edit the DNA coding strand in the editor.
- Point out letters are bases and color-coded.

## 3) Replication (60–90s)
- Click **Replication**.
- Explain base pairing: A↔T, C↔G.
- Show complementary strand.

## 4) Transcription (60–90s)
- Click **Transcription**.
- Point out RNA uses **U** instead of T.
- Show mRNA and codon grouping.

## 5) Translation (90–120s)
- Click **Translation**.
- Find `AUG` start and show amino acid chain grows.
- Explain stop codon ends the protein.

## 6) Fun “mutation” moment (30–60s)
- Change 1 DNA base.
- Show how mRNA/codons/protein change.
